Resources

This page keeps the reference list small. The guide explains the needed Slurm ideas through local examples and CSD3 commands, so the links below are for checking official details later.

CSD3 Ground Truth

CSD3 Quick Start is the starting point for new users. Use it to confirm account basics and the official CSD3 orientation.

Connecting to CSD3 covers SSH access. Use it when local SSH or Remote-SSH cannot connect.

Running Jobs on CSD3 is the official reference for CSD3 batch jobs. Use it to check scheduler terms, partitions, and job submission behavior.

Login-Web Interface is the fallback path when you need browser-based shell access.

VS Code Remote Work

Remote Explorer is the VS Code view that shows configured SSH targets.

Python Workflow References

uv is the Python environment and package tool used in the setup example.

Python venv explains the standard-library virtual environment concept behind isolated project environments.

Python argparse tutorial is useful when turning a local script into a command that handles one unit of work at a time.

Papermill is relevant for notebook-based workflows and the notebook helper scripts in this repo.
